Pipeline de candidaturas

Lanza ahora mismo la búsqueda de ofertas y el envío del reporte a tu correo.

Cómo crear el token (una sola vez): github.com/settings/personal-access-tokens/new → Repository access: Only select repositories → auto-candidaturas → Permissions → Repository permissions → Actions: Read and write → Generate token.

⚠️ El token da control sobre las ejecuciones de tu repositorio: no compartas este archivo con el token guardado ni lo subas a un repositorio público. El reporte tarda en llegar unos 5–8 minutos tras pulsar el botón.
